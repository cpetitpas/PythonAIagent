Using PAI to Answer Questions

  • PAI leverages advanced AI models to understand your questions.
  • PAI searches through a local database built on your machine from PDF files that you upload to answer your questions about your Social Security, healthcare records, and much more.
  • PAI runs on your local machine, ensuring data privacy and security. At no time is your personal information uploaded to external servers.
  • When you stop PAI, your local database is NOT kept. When you start PAI again, you will have to reload your PDF files. This further protects your personal/sensitive information.
  • PAI is designed to be user-friendly, with a simple interface that makes it easy to ask questions and get answers quickly.
  • To Use PAI:
    • You will need an OpenAI API key. Follow these steps to get started:
      • Sign up for an account on the OpenAI website (https://www.openai.com/).
      • Create a new API key in your account settings.
      • Copy the API key and paste it into the PAI configuration file (.env) on your local machine.
      • Purchase credits from OpenAI to use the API. Only $5 worth will get you started. PAI allows you to choose the Embedding Model and the Answer Model that best fits your needs.
    • Start the PAI application.
    • Select Choose Files and select the PDF files you want to upload.
    • Click Upload Selected to load the files into your local database. Select Clear My Info to clear your uploaded files and start fresh.
    • Once the files are uploaded, you can ask questions in the "Chat with PAI" section.
    • IMPORTANT: PAI uses your local database to answer questions. Make sure you type names correctly.
